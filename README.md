# odin-recipes

## Description
This project is part of the Foundations course at The Odin Project. It is a basic recipe website built using HTML. The goal is to practice setting up a project structure and using Git/GitHub effectively.

## Skills Demonstrated
By the end of this project, I will have demonstrated the following:
* Creating and managing a Git repository.
* Basic HTML structure (headings, paragraphs, links, and images).
* Organizing files and directories within a project.
* Using the Linux terminal to navigate and manipulate files.
